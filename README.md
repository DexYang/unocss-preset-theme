# unocss-preset-theme

This preset will help you easily make dynamic theme switching.

## Installation

```bash
npm i -D unocss-preset-theme
```

```ts
import presetTheme from 'unocss-preset-theme'

Unocss({
  presets: [
    presetTheme({
      theme: {
        dark: {
        },
        light: {
        },
        compact: {
        }
      }
    }),
  ],
})
```
